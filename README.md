# GradeCard

Simple C program to input student marks (5 subjects), calculate total, percentage and grade,
display the result and append it to `grades.txt`.

## How to use
1. Compile: `gcc grade_card.c -o grade_card`
2. Run: `./grade_card` (on Windows use `grade_card.exe`)
3. Enter name, roll and 5 marks.
4. Result shown on screen and saved to `grades.txt`.

## Files
- `grade_card.c` : source code
- `grades.txt`   : created after first run, contains saved student records (CSV)
